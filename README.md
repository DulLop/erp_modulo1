# ERP Módulo "Inventario de Materia Prima"

Primer Módulo de un sistema ERP modular desarrollado con:

- PHP 8
- Laravel
- MySQL
- Tailwind CSS

## Objetivo
Desarrollar un sistema escalable para gestión de procesos internos,
iniciando con el módulo de Inventario de Materia Prima.

## Estado
Módulo en desarrollo (entorno local).
